O que é HTML?

HTML significa linguagem de marcação de hipertexto. Ela é a linguagem de marcação padrão para criação de páginas da Web, descrevendo a estrutura de uma página. HTML consiste em uma série de elementos que informam ao navegador como exibir o conteúdo para o usuário do site, ele que rotula partes do conteúdo como “este é um título”, “este é um parágrafo”, “este é um link”, etc.

Apresentação das Tags HTML:

  • < h1> - este elemento que define um título grande.
    exemplo:

    Título de ordem maior

  • < p> - este elemento que define um parágrafo.
    exemplo:

    Parágrafo. Lorem ipsum dolor, sit amet consectetur adipisicing elit. Culpa repellat natus eaque sint distinctio ipsa dolorum eius, pariatur libero molestiae atque omnis repudiandae non illum velit inventore dolore temporibus quae!

  • < ol> - este elemento que define uma lista ordenada.
    exemplo:
    1. Item 1
    2. Item 2
    3. Item 3
  • < ul> - este elemento que define uma lista não ordenada.
    exemplo:
    • Item 1
    • Item 2
    • Item 3
  • < strong> - este elemento que define uma parte do texto delimitada para negrito.

    exemplo:
    Lorem ipsum dolor sit amet consectetur adipisicing elit. Adipisci perspiciatis aut rsequi consequuntur ipsum earum fugiat perferendis, eos nihil. Aut commodi perferendis, repellendus ex impedit exercitationem quam architecto labore ad.

  • < u> - este elemento que define uma parte do texto delimitada para sublinhada.

    exemplo:
    Lorem ipsum dolor sit amet consectetur adipisicing elit. Suscipit quos rerum fugit odio culpa sapiente voluptatibus debitis perspiciatis non atque officiis ullam, sint omnis? Molestiae consectetur aliquid non quod eveniet!

  • < mark> - este elemento que define uma parte do texto delimitada para marcada.

    exemplo:
    Lorem ipsum dolor, sit amet consectetur adipisicing elit. Asperiores culpa corrupti dolores corporis alias non facere eos accusamus iure hic sint sed tempora, beatae blanditiis consequatur cum voluptatum at ex.

  • < a> - este elemento que define um link dentro de seu web site.
    exemplo:

    Clique aqui para abrir a página do Youtube.
